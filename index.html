📝 学习日志

记录每一步的成长，哪怕很小

2026 年 5 月 30 日
🌐 建站

我的第一个网站上线了！

今天我用 GitHub Pages 部署了我的第一个个人网站！ 整个过程比我想象的简单太多了——在 GitHub 上创建仓库、 把写好的 HTML 文件传上去、开启 Pages，几分钟网站就上线了。 虽然现在还很简单，但这是第一步！🎉

2026 年 5 月 29 日
🐙 GitHub

学会了 GitHub Desktop 的基本操作

下载了 GitHub Desktop，终于不用记那些复杂的命令行啦。 学会了 Clone（下载仓库）、Commit（保存版本）、Push（上传云端） 这三个核心操作。用一键汉化工具把它变成了中文界面，方便多了。

2026 年 5 月 28 日
🤖 AI 学习

用 AI 帮我写代码，原来这么简单

今天让 Claude 帮我生成了一个 HTML 网页文件。我什么代码都没写， 就告诉 AI"我想要一个个人介绍页面"，它就给我生成了完整的代码。 我复制保存，上传到 GitHub，网站就出来了！ 这种感觉太神奇了，科技真的改变了学习的方式。

2026 年 5 月 27 日
🐙 GitHub

注册了 GitHub 账号，开始了第一步

今天注册了我的第一个 GitHub 账号，创建了第一个仓库。 虽然还不太明白这些东西具体是做什么的，但至少迈出了第一步。 看了教程，大概理解了"仓库"就像是一个项目的文件夹， "提交"就是保存一个版本。慢慢来吧！
